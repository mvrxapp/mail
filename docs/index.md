# AECS

AECS (AI Email Consumption) is a specification for representing a parsed email
message — `NormalizedEmail` — in a form suitable for AI/LLM consumption, storage,
and threading. This site hosts the specs and links to the reference implementation.

- [AECS-1 specification](./specs/aecs-1.html) — the normative `NormalizedEmail` spec
- [AECS-SDK-1 specification](./specs/aecs-sdk-1.html) — the target TypeScript SDK surface
- [NormalizedEmail JSON Schema](https://github.com/mvrxapp/mail/blob/main/packages/aecs/specs/schema/normalized-email.schema.json)
- [Conformance suite](https://github.com/mvrxapp/mail/blob/main/packages/aecs/specs/conformance)
- [`@mvrx/aecs`](https://www.npmjs.com/package/@mvrx/aecs) on npm
- [`@mvrx/mail`](https://www.npmjs.com/package/@mvrx/mail) on npm
